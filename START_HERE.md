## 🚀 Quick Start Options

### Option 1: Docker (5 minutes) ⚡
```bash
cd /Users/somanigunni/Farming
docker-compose up
```
Everything runs: backend, database, redis

### Option 2: Manual Setup (15 minutes) 🔧
```bash
# Backend
cd backend && npm install && npm run dev

# Mobile (new terminal)
cd mobile && npm install && npm start
```
